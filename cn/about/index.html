关于

我是一个啥都来点但啥也没做好的混混，想法很多，但付诸行动的极少，写个博客都非常不上心，但是换博客的平台还算勤快。我从 2005 年开始在 MSN Space 尝试用博客平台，中间用过一段时间 Q-zone，在 Wordpress 被墙之前也尝试过一下，Live Space 关闭以后，在 163 Blog 上流浪了一段时间，Blogcn 升级成 Wordpress 平台后又将所有日志迁移过去，辗转八年后我到了码农的乐土 Jekyll，但 Jekyll 的龟速和麻烦的配置一直困扰着我，几经波折后来到了 Hugo 的怀抱1。到目前为止，我对 Hugo 在速度方面表现非常满意，但感觉功能方面比 Hexo 还稍差些2，希望 Hugo 能够成为我的最后一站。

在选择博客的名字时，想来想去只想起了最喜欢的歌星的一首歌：

我想了半天
歌手：张德兰
专辑：寂寞是我
词：郑国江

念到那次相对默无言
爱的感觉淡淡然
但为何在我脑海盘旋
牵起丝丝挂牵
我想了半天 又挂念半天
只感到他的一切淡如梦里的轻烟
轻轻呀飘到面前 又似看不见
念到那次一切在目前
眼中的你默默然
但热情在你眼中可见 不用明言
我想了半天 又纪念半天
挥不去他的一切 但求愿看清一点
只想飘到你面前 让我见多见

既然想了半天也想不出个好听的名字，干脆就用这歌的名字当博客名吧。

音乐地址：虾米

  1. 说实话，Jekyll 和 Hexo 都不适合普通的博客用户，Hugo 虽然配置简单到了极限，但实现各种自定义的功能同样需要了解非常多的东西，并且需要花费非常多的精力，普通用户还是老老实实用 Wordpress 比较好。 ↩
  2. 经过几天的努力，通过一些泼皮但基本可以凑效的做法，实现了好几种比较看重的功能，如 Section 下的分类列表和标签云，本地搜索以及图片浏览功能。可惜的是，在 Markdown 引擎方面，Hugo 还没有其它的选择。 ↩
